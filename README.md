# dotfiles

Prime the system:
```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/Raiu/dotfiles/main/prime.sh)"
```

install dotfiles
```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/Raiu/dotfiles/main/install.sh)"
```


Exa
/usr/local/bin
/usr/local/share/man
/usr/local/share/zsh/site-functions
